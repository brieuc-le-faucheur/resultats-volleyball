# Volleyball Dashboard

Dashboard pour consulter les résultats et classements de volleyball.

POC créé pour expérimenter le vibe coding avec Claude Code.

## Fonctionnalités

- Affichage des classements avec statistiques complètes
- Liste des matchs regroupés par date (joués / à venir)
- Filtrage par équipe (clic sur une ligne du classement)
- Thème clair/sombre
- Interface responsive

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- Vue 3 (Composition API)
- TypeScript
- Vite

## Source des données

Les données affichées proviennent du site officiel de la [FFVB](https://www.ffvb.org/) (Fédération Française de Volleyball) et sont récupérées en temps réel via leur site public.

Ce projet est un outil personnel non affilié à la FFVB, propriétaire des données.
